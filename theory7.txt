q3.âœ… Scope of Variables in Python

Scope refers to the region of a program where a variable is recognized and can be used.

Python uses the LEGB Rule to determine variable scope:

L â†’ Local
E â†’ Enclosing
G â†’ Global
B â†’ Built-in

Let's break these down.

ğŸ”¹ 1. Local Scope (L)

A local variable is defined inside a function and can only be accessed there.

def func():
    x = 10  # local variable
    print(x)

func()
print(x)  # âŒ Error: x is not defined outside the function


âœ” Exists only inside the function
âœ” Created when the function is called
âœ” Destroyed when the function exits

ğŸ”¹ 2. Enclosing Scope (E)

Variables in the outer function of a nested function.

def outer():
    y = 20  # enclosing variable

    def inner():
        print(y)  # accessible inside nested function

    inner()

outer()


âœ” Used in nested functions
âœ” Not the local scope or global scope
âœ” Also called nonlocal scope

You can modify an enclosing variable using nonlocal:

def outer():
    x = 5
    def inner():
        nonlocal x
        x = 10
    inner()
    print(x)  # 10

ğŸ”¹ 3. Global Scope (G)

A global variable is defined outside all functions and can be accessed anywhere in the file.

x = 100  # global variable

def show():
    print(x)

show()
print(x)

Modifying a global variable inside a function:

Use the global keyword:

x = 10

def modify():
    global x
    x = 20

modify()
print(x)  # 20

ğŸ”¹ 4. Built-in Scope (B)

Variables that come from Python's standard library.

Examples:

print()
len()
range()


These names exist in Pythonâ€™s built-in namespace, always available unless redefined.

ğŸ” LEGB Rule Summary

When Python sees a variable, it searches in this order:

Local â†’ inside current function

Enclosing â†’ in outer function(s)

Global â†’ defined at top level of file

Built-in â†’ Python-provided names

Example:

x = "global"

def outer():
    x = "enclosing"
    def inner():
        x = "local"
        print(x)
    inner()

outer()


Output:

local


Python picks the first match in the LEGB order.

ğŸ“Œ Final Summary
Scope	Where defined	Accessible where?
Local	Inside a function	Only in that function
Enclosing	Outer function of nested functions	Inner functions
Global	Top-level of the script	Entire module
Built-in	Python built-ins	Everywhere
