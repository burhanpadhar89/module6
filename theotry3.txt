q2. Pythonvariables and memory allocation.
üß© 1. What Is a Variable?

A variable is a name that refers to a value stored in memory.
In Python, variables are references (labels), not actual boxes holding data like in some other languages (e.g., C or Java).

‚úÖ Example:

x = 10


x is a variable name (a label).

10 is an integer object stored somewhere in memory.

x points (or refers) to that memory location.

üß† 2. How Memory Allocation Works

When you assign a value to a variable:

Python creates an object in memory (if it doesn‚Äôt already exist).

The variable name refers to (points to) that object.

Example:
a = 5
b = a


Here‚Äôs what happens:

Python creates an integer object 5 in memory.

a refers to that object.

When you write b = a, both a and b point to the same memory location.

üß© Memory diagram:

a ‚îÄ‚îÄ‚ñ∫ [ 5 ]
b ‚îÄ‚îÄ‚îò


If you later change a:

a = 10


Python creates a new object (10) in memory, and now a points to it ‚Äî but b still points to 5.

a ‚îÄ‚îÄ‚ñ∫ [ 10 ]
b ‚îÄ‚îÄ‚ñ∫ [ 5 ]

‚öôÔ∏è 3. Mutable vs Immutable Objects

This concept is crucial for understanding how memory behaves.

Type	Mutable?	Examples
Immutable	‚ùå No (cannot change value in place)	int, float, str, tuple
Mutable	‚úÖ Yes (can change value in place)	list, dict, set
Example ‚Äî Immutable (Integers):
x = 5
y = x
x = x + 1

print(x)  # 6
print(y)  # 5


üîπ A new object 6 is created; x now refers to it.
üîπ y still points to 5.

Example ‚Äî Mutable (Lists):
list1 = [1, 2, 3]
list2 = list1
list1.append(4)

print(list1)  # [1, 2, 3, 4]
print(list2)  # [1, 2, 3, 4]


üîπ Both list1 and list2 refer to the same list object in memory, so modifying one affects the other.

üßÆ 4. Memory Management in Python

Python uses automatic memory management, mainly through:

Reference counting (keeps track of how many variables reference an object).

Garbage collection (frees memory when no references exist).

Example:
x = [1, 2, 3]
y = x
del x  # removes one reference

# The list still exists because 'y' points to it
print(y)  # [1, 2, 3]

del y  # removes the last reference
# Now, the list is garbage-collected automatically

üß† 5. The id() Function

You can use id() to check the memory address of an object.

a = 100
b = 100
print(id(a), id(b))  # Same id (Python caches small integers)

x = [1, 2, 3]
y = [1, 2, 3]
print(id(x), id(y))  # Different ids (new list objects)

üîç 6. Python‚Äôs Object Caching (Interpreted Optimization)

Python interns (reuses) some immutable objects for efficiency:

Small integers (from -5 to 256)

Some short strings

That‚Äôs why:

a = 100
b = 100
print(a is b)  # True (same memory)


But:

a = 1000
b = 1000
print(a is b)  # False (different memory locations)

üß© 7. Summary Table
Concept	Description	Example
Variable	Name referring to an object in memory	x = 10
Immutable Object	Value can‚Äôt change; reassignment creates a new object	int, str, tuple
Mutable Object	Value can be changed in place	list, dict, set
Reference Counting	Python tracks how many variables point to an object	del decreases count
Garbage Collection	Frees memory when no references remain	Automatic process

q3. Pythonoperators: arithmetic, comparison, logical, bitwise
üßÆ 1. Arithmetic Operators

Used to perform mathematical calculations between numeric values (int or float).

Operator	Description	Example	Result
+	Addition	5 + 3	8
-	Subtraction	10 - 4	6
*	Multiplication	7 * 3	21
/	Division (float result)	10 / 4	2.5
//	Floor Division (integer result)	10 // 4	2
%	Modulus (remainder)	10 % 4	2
**	Exponentiation (power)	2 ** 3	8

‚úÖ Example:

a = 10
b = 3

print(a + b)   # 13
print(a - b)   # 7
print(a * b)   # 30
print(a / b)   # 3.333...
print(a // b)  # 3
print(a % b)   # 1
print(a ** b)  # 1000

‚öñÔ∏è 2. Comparison (Relational) Operators

Used to compare two values ‚Äî results are Boolean (True or False).

Operator	Description	Example	Result
==	Equal to	5 == 5	True
!=	Not equal to	5 != 3	True
>	Greater than	7 > 5	True
<	Less than	2 < 4	True
>=	Greater than or equal to	5 >= 5	True
<=	Less than or equal to	3 <= 2	False

‚úÖ Example:

x = 10
y = 20

print(x == y)  # False
print(x != y)  # True
print(x > y)   # False
print(x < y)   # True
print(x >= 10) # True
print(y <= 15) # False

üß† 3. Logical Operators

Used to combine conditional statements ‚Äî evaluate expressions that return True or False.

Operator	Description	Example	Result
and	True if both operands are True	(x > 5 and x < 15)	True
or	True if either operand is True	(x > 15 or y > 15)	True
not	Reverses the result	not(x > 5)	False

‚úÖ Example:

x = 10
y = 20

print(x > 5 and y > 15)  # True
print(x > 15 or y > 15)  # True
print(not (x > 5))       # False

‚öôÔ∏è 4. Bitwise Operators

Used to perform operations on binary (bit) representations of integers.

Operator	Description	Example	Result (Binary)	Result (Decimal)
&	Bitwise AND	5 & 3	0101 & 0011 = 0001	1
`	`	Bitwise OR	`5	3`
^	Bitwise XOR	5 ^ 3	0101 ^ 0011 = 0110	6
~	Bitwise NOT	~5	~0101 = -(0101 + 1)	-6
<<	Left shift	5 << 1	0101 ‚Üí 1010	10
>>	Right shift	5 >> 1	0101 ‚Üí 0010	2

‚úÖ Example:

a = 5   # (binary 0101)
b = 3   # (binary 0011)

print(a & b)   # 1
print(a | b)   # 7
print(a ^ b)   # 6
print(~a)      # -6
print(a << 1)  # 10
print(a >> 1)  # 2
